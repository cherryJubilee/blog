# 내장 객체

<!-- more -->

## 내장 객체(implicit Object)

1. 요청과 응답, 화면 출력 등 웹 프로그래밍에 있어 필수로 사용하는 객체
2. 특징
    - 컨테이너가 사용합
    - 별도의 객체 생성 없이 내장 객체의 메서드 사용 가능

## request 객체

1. 클라이언트가 전송한 요청 정보를 담고 있는 객체
2. 요청 헤더 밑 쿠키 정보 읽기
