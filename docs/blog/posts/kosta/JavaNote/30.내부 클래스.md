## 💡 내부클래스 개념

## 💡 이너클래스 종류

```java
class A { // 아우터 클래스

    class B { //

    }

}
```

## 💡 객체 생성하기

1. 먼저 아우터 클래스의 객체를 생성
2. 이후 생성한 아우터 클래스 객체의 참조 변수를 이용해 객체 내부에 있는 이너 클래스의 생성자 호출함
3. 문법

    ```java
    아웃 클래스 아웃 클래스 참조 변수 = new 아웃 클래스();
    아웃 클래스.이너 클래스 이너 클래스 참조 변수 =  아웃 클래스 참조 변수.new 이너 클래스();


    class A{  // 아우터 클래스
        class B { // 이너 클래스

        }
    }

    A a = new A();
    A.B b = new a.B();
    ```
